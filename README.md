# Technical Communication & Infrastructure Portfolio

Welcome! This repository serves as a **Technical Sales Enablement & Software Engineering Showcase**. It bridges the gap between deep technical concepts, structural design, and human-centric technical communication. 

As a Computer Engineering student at **Universidad de Guadalajara (CUCEI)** with a multi-disciplinary background in teaching, public presentation, and competitive programming, my core driving narrative is:
> *"I thrive on mastering complex, abstract technologies and translating them into clear, actionable, and visually compelling narratives for diverse audiences—from technical peers to enterprise stakeholders."*

## 🛠️ Portfolio Architecture

This portfolio is divided into two distinct strategic domains designed to showcase architectural depth, infrastructure familiarity, and technical storytelling using generative AI workflows.

1. **[Enterprise Infrastructure & Cybersecurity](#1-enterprise-infrastructure--cybersecurity-showcase)**: Highlighting data center fundamentals, network protocols, and security vector breakdowns.
2. **[Software Engineering & System Design (BarberOS Project)](#2-software-engineering--system-design-barberos-showcase)**: Showcasing full-lifecycle software modeling, interaction flowcharts, and system architecture mapping.
